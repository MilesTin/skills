# Stock Screening - 选股技能

本技能用于筛选高股息率、低市盈率的股票，并通过分析财报PDF计算实际分红率。

## 工作流程

### 1. 粗选股票

mcporter调用 `screen_predefined` 工具进行初步筛选：
- 股息率 >= 8%
- 市盈率 > 0 且 <= 10

### 2. 获取财报数据

对筛选出的每只股票：
1. 使用 `yfinance` 获取最近一年的财报数据
2. 获取当前市值（market cap）

### 3. 下载并分析财报PDF

1. 下载对应公司的财报PDF，可以是年报或中报（10-K/年报 或 10-Q/中报）
   - **必须满足**：报告发布时间在最近一年内
2. 读取PDF内容，提取以下信息：
   - **分红部分**：找到总分红金额（Dividends/分红总额）
   - **股本部分**：找到当前总股份和去年同期总股份

### 4. 计算实际分红率

**步骤1：计算真实股息率**
```
真实股息率 = 总分红金额 / 当前市值
```

**步骤2：计算股份减少比例**
```
股份减少比例 = 1 - (当前总股份 / 去年同期总股份)
```
- 如果股份增加，此值为负数
- 如果股份减少，此值为正数（表示公司回购股份）

**步骤3：计算实际分红率**
```
实际分红率 = 真实股息率 + 股份减少比例
```

### 5. 输出结果

为每只股票输出：
- 股票代码
- 公司名称
- 当期财报
- 去年同期财报
- 当前市值
- 总分红金额
- 真实股息率 (%)
- 去年同期总股本
- 当前总股本
- 股份减少比例 (%)
- **实际分红率 (%)** - 核心指标

## 注意事项

1. **PDF解析**：财报PDF结构可能复杂，注意处理：
   - 表格数据的分页问题
   - 中英文混杂的情况
   - 不同公司的财报格式差异

2. **数据验证**：
   - 检查分红金额是否与市值匹配（避免数量级错误）
   - 验证股份数据的合理性
   - 对比yfinance数据和PDF数据的一致性，如果不一致，**以当期财报数据为准**

3. **特殊情况处理**：
   - 如果找不到分红数据，标记为N/A
   - 如果股份数据缺失，股份减少比例记为0
   - 对于新上市公司，可能没有去年同期数据

## 依赖工具

- `screen_predefined`: 股票筛选工具
- `yfinance`: 获取股票基本信息和财报元数据
- PDF读取工具：提取财报PDF内容
