# Stock Valuation - 股票估值技能

本技能提供多种股票估值方法，帮助判断股票的内在价值和投资潜力。

## 估值方法

### 1. DCF现金流折现模型（推荐用于成长型公司）

**适用场景**：稳定盈利、现金流可预测的公司

**计算公式**：
```
企业价值 = Σ [FCF_t / (1+r)^t] + 终值 / (1+r)^n

其中：
- FCF_t = 第t年的自由现金流
- r = 折现率（通常8%-12%）
- n = 预测期（通常5-10年）
- 终值 = FCF_n * (1+g) / (r-g) （g为永续增长率）
```

**数据需求**：
- 历史自由现金流（3-5年）
- 营收增长率预测
- 净利润率
- 资本支出

---

### 2. PE/PB相对估值法（推荐用于成熟公司）

**PE市盈率法**：
```
合理股价 = 每股收益(EPS) × 行业平均PE
或
合理PE = 当前PE × (预期增长率 / 100)
目标价 = EPS × 合理PE
```

**PB市净率法**：
```
合理股价 = 每股净资产(BPS) × 行业平均PB
```

**数据需求**：
- 当前PE、PB
- 行业平均PE、PB
- 每股收益(EPS)、每股净资产(BPS)
- 预期增长率

---

### 3. 格雷厄姆估值法（安全边际法）

**适用场景**：寻找有安全边际的价值股

**计算公式**：
```
内在价值 = EPS × (8.5 + 2g) × (4.4 / AAA债券利率)

其中：
- EPS = 每股收益
- g = 预期增长率（%）
- 4.4 = 格雷厄姆时期的无风险利率基准
```

**安全边际**：
```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%
建议：安全边际 > 30% 才考虑买入
```

---

### 4. 股息折现模型（DDM，推荐用于高股息股）

**适用场景**：稳定分红的高股息股票

**戈登增长模型**：
```
内在价值 = D₁ / (r - g)

其中：
- D₁ = 下一年预期股息 = D₀ × (1+g)
- r = 要求回报率（折现率）
- g = 股息永续增长率（通常2%-5%）
```

**两阶段DDM**（适用于增长率变化的公司）：
```
价值 = 高增长阶段现值 + 稳定增长阶段现值
```

---

### 5. EV/EBITDA企业价值倍数法

**适用场景**：资本结构差异大的公司对比

**计算公式**：
```
企业价值(EV) = 市值 + 总债务 - 现金
EV/EBITDA = 企业价值 / EBITDA

目标股价 = (行业平均EV/EBITDA × 公司EBITDA - 债务 + 现金) / 总股本
```

## 工作流程

### 1. 数据收集

使用 `yfinance` 获取：
- 当前股价、市值
- 财务报表（利润表、资产负债表、现金流量表）
- 历史PE、PB
- 股息历史

### 2. 选择估值方法

根据股票类型选择：
| 股票类型 | 推荐方法 | 次选方法 |
|---------|---------|---------|
| 成长股 | DCF | PEG |
| 价值股 | 格雷厄姆法 | PE/PB |
| 高股息股 | DDM | DCF |
| 周期性股 | PB | EV/EBITDA |
| 金融股 | PB | PE |

### 3. 计算与验证

1. 使用选定方法计算内在价值
2. 与当前股价对比，计算上涨/下跌空间
3. 使用第二种方法交叉验证
4. 进行敏感性分析（关键参数变化对结果的影响）

## 注意事项

1. **估值不是预测**：估值是基于假设的估算，不是对未来股价的保证
2. **多种方法结合**：单一估值方法可能失真，建议至少使用2-3种方法
3. **定性因素**：估值模型无法涵盖管理层质量、竞争优势、行业趋势等定性因素
4. **定期更新**：估值结果应随季报/年报发布而更新
5. **保守假设**：在不确定时，使用保守假设（较低的增速、较高的折现率）
